Definición de HTML5

Podemos definir HTML5 como un estándar que sirve para
definir la estructura y el contenido de una página Web.

Segun Wikipedia

HTML5 es un estándar que sirve como referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones,
define una estructura básica y un código (denominado HTML) para la definición de contenido de una página web,
